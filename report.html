问题1 - 特征观察¶

如前文所述，本项目中我们关注的是其中三个值:'RM'、'LSTAT' 和'PTRATIO'，对每一个数据点:

  • 'RM' 是该地区中每个房屋的平均房间数量；
  • 'LSTAT' 是指该地区有多少百分比的房东属于是低收入阶层（有工作但收入微薄）；
  • 'PTRATIO' 是该地区的中学和小学里，学生和老师的数目比（学生/老师）。

凭直觉，上述三个特征中对每一个来说，你认为增大该特征的数值，'MEDV'的值会是增大还是减小呢？每一个答案都需要你给出理由。

提示：你预期一个'RM' 值是6的房屋跟'RM' 值是7的房屋相比，价值更高还是更低呢？

回答:

  • RM:直观上而言，若是同样面积的房屋，那么每个房屋的平均房间数量越多，则相应的每个房间的面积就小了，所以如果在房屋面积一定的前提下，我倾向于认为RM越高房屋价值越低；但是如果认为平均房间面积一定的话，那么房间越多，自然房屋也就越大，房屋价格也就越贵，所以按照平常接触到的所谓“一室一厅”、“两室一厅”的概念而言，我倾向于后者，即RM越高，房屋价格越高；
  • LSTAT:如果一个地方低收入阶层站的比重高，那么可以认为该地区房租、房价如果能让大多数人接受的起，那么相应的房屋价格应该比较低，所以我主观认为LSTAT越高，房屋价格越低；
  • PTRATIO: 学生和老师的数目比可以作为衡量该地区的教学质量的一个参数，可以认为一个学生可分配的教师资源越多，该地区的教育质量越高相应的房价也就越高（学区房嘛），所以换言之，PTRATIO越高，房屋价格越低；

练习：定义衡量标准¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

from sklearn.metrics import r2_score

def performance_metric(y_true, y_predict):
    """ Calculates and returns the performance score between 
        true and predicted values based on the metric chosen. """
    
    
    score = r2_score(y_true,y_predict)
    
    
    return score

什么是网格搜索法？如何用它来优化学习算法？

回答:

  • 网格搜索法是超参数优化的一种常用方法，可以通过GridSearchCV来系统地遍历多种参数组合，通过交叉验证确定最佳效果参数；
  • 类似一种穷举的暴力方法将所有指定的参数组合遍历一遍并通过指定的评判标准进行量化来从中挑选出得分最高的参数组合，所以通过网格搜索法可以帮你快速判断所使用的算法对于当前数据最适用的参数组合

问题 8- 交叉验证¶

什么是K折交叉验证法（k-fold cross-validation）？优化模型时，使用这种方法对网格搜索有什么好处？网格搜索是如何结合交叉验证来完成对最佳参数组合的选择的？

提示： 跟为何需要一组测试集的原因差不多，网格搜索时如果不使用交叉验证会有什么问题？GridSearchCV中的'cv_results'属性能告诉我们什么？

答案:

  • K折交叉验证法是一种在对测试集进行测试之前先在训练集上对指定算法的参数进行评估的方法，具体做法是将训练集分为k组，然后进行k次模型评估，每次评估以其中一组数据作为评估数据而另外的k-1组数据作为当前的训练数据，最后对k组数据的评估结果取平均值来衡量评估结果
  • 将K折交叉验证法与网格搜索相结合可以针对特定的算法在对测试机进行测试之前进一步分析该算法中的可能参数组合中的最佳组合，从而进一步优化算法，取得最佳模型；
  • 两者结合类似用一种穷举的暴力方法将某个算法的所有指定的参数组合遍历一遍并通过指定的评判标准进行量化，最后从中挑选出得分最高的参数组合

    grid = GridSearchCV(estimator=regressor,param_grid=params,scoring=scoring_fnc,cv=cv_sets)

    # Fit the grid search object to the data to compute the optimal model
    grid = grid.fit(X, y)

    # Return the optimal model after fitting the data
    return grid.best_estimator_

做出预测¶

当我们用数据训练出一个模型，它现在就可用于对新的数据进行预测。在决策树回归函数中，模型已经学会对新输入的数据提问，并返回对目标变量的预测值。你可以用这个预测来获取数据未知目标变量的信息，这些数据必须是不包含在训练数据之内的。
